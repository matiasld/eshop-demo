# eshop-demo
## E-Shop Mock-up:
Android + Java
  
Features:  
- Login/User Creation screen  
- Authentication via E-Mail  
- Profile preferences  
- Realtime database  
- Favorites tab  
*(Authentication, RT-Database implemented with Google Firebase)
